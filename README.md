# 🎬 Movie Recommendation System

A **Machine Learning based Movie Recommendation System** that suggests movies to users based on similarity between movie features.  
This project is my **first Machine Learning project**, built to understand the fundamentals of recommendation systems, text preprocessing, and similarity-based models.

---

## 📌 Features

- Recommend movies based on a selected movie
- Uses **content-based filtering**
- Computes similarity between movies
- Clean preprocessing of movie metadata
- Simple and fast recommendation logic

---

## 🧠 ML Concepts Used

- Natural Language Processing (NLP)
- Text preprocessing
- Feature extraction
- Cosine similarity
- Vectorization (Bag of Words / CountVectorizer)

---

## ⚙️ Tech Stack

- **Python**
- **Pandas**
- **NumPy**
- **Scikit-learn**
- **NLTK**
- **Jupyter Notebook**

## 🚀 How It Works

1. Load movie dataset
2. Preprocess movie metadata (genres, keywords, cast, crew)
3. Convert text data into vectors using **CountVectorizer**
4. Compute **cosine similarity** between movie vectors
5. Recommend the most similar movies



## 📊 Dataset

Dataset used: **movies.csv**

It contains information like:
- movie title
- genres
- cast
- crew
- keywords
- overview
- ...
